## 核心约束规则

前端的 system prompt 包含以下关键约束（现在已正确生效）：

### 仿写爆款任务
1. **严禁使用业务关键词**（数字资产、AI获客、内容增长等）
2. **严禁编造人物**（马斯克、刘强东、李亚鹏等）
3. **必须基于参考文案**的核心元素：
   - 时间元素（如"再过五年"、"2025年"）
   - 数字元素（如"三样东西"）
   - 结论元素（如"买不起"）
4. **保留核心命题**，只改变表达方式

### 钩子（皮）的标准
- 第一句话就能抓、爆料、命令、预警
- 不是解释或总结
- 调动用户情绪

## 常见问题

### Q: 生成的钩子还是与原文无关？
A: 检查以下几点：
1. 后端服务是否重启（必须重启才能加载新代码）
2. 前端是否刷新页面
3. "使用实时 API"开关是否打开
4. 查看浏览器控制台是否有错误

### Q: API 调用失败？
A: 检查：
1. 后端服务是否运行（访问 http://127.0.0.1:8787/api/health）
2. API Key 是否配置正确
3. 网络连接是否正常

### Q: 端口冲突？
A: 当前统一使用 `8787`，优先检查 `backend/config.local.json` 或 `server/config.local.json`

## 技术架构

```
前端 (React + TypeScript)
  ↓ HTTP POST /api/chat/completions
统一后端 (FastAPI, 8787)
  ↓ 直接转发（不修改 messages）
第三方 API (Gemini)
  ↓ 返回 JSON
后端
  ↓ 直接返回
前端
  ↓ 解析并显示
```
